# MovieSearch

Front-end приложение для поиска и просмотра фильмов. Демо: https://movie-search-ms.vercel.app/

# Стек технологий:

-   React + React Hooks
-   TypeScript
-   Redux Toolkit, RTK Query
-   Sass, CSS modules
-   Jest
-   NextJS
-   PWA
-   React-hook-form
-   Storybook
