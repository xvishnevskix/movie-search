# Kinomore
 Junior Front-end приложение для поиска и просмотра фильмов. Демо: [https://kinomore.vercel.app/](https://kinomore.vercel.app/)
 
# Стек технологий:
  * React + React Hooks
  * TypeScript
  * Redux Toolkit, RTK Query
  * Sass, CSS modules
  * Jest
  * NextJS
  * PWA
  * React-hook-form
  * Storybook
